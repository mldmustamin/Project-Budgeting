# FMv2 Repo Audit

Use this skill when you need to understand the current repository state before making changes.

## Read first
- docs/00_REPO_AUDIT.md
- docs/03_ARCHITECTURE.md
- docs/14_RELEASE_CHECKLIST.md

## What to do
- Confirm the relevant module, package, and layer before editing.
- Identify files that are safe to change.
- Note mismatches between docs and code.
- Prefer no-code recommendations if the task is still exploratory.

## Output format
- Current state
- Risks
- Files to inspect next
- Safe next action
