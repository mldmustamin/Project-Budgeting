# Report and Export

## Current Export Capabilities (Android)

| Format | Implementation | Entry Point |
|--------|----------------|-------------|
| PDF | `ReportFileRepositoryImpl.createPdf()` | Project dashboard share action |
| Excel (XLSX) | `ReportFileRepositoryImpl.createExcel()` | Project dashboard share action |
| CSV | `ExportCsvUseCase.execute()` | Project dashboard / ViewModel |

All exports driven by `PrepareProjectReportUseCase` which uses `CalculateProjectSummaryUseCase` for totals.

---

## Export Pipeline

```
ProjectDashboard ViewModel
  → PrepareProjectReportUseCase(projectId)
      → CalculateProjectSummaryUseCase  (SSOT totals)
      → FundsRepository (transactions, accounts, categories)
      → ProjectReportData
  → ReportFileRepository.createPdf/Excel
     or ExportCsvUseCase.execute
  → File in cache/reports/ or CSV string
  → Android Share Intent (FileProvider)
```

---

## PDF Contents

- Header: project name, period, export timestamp, transaction count
- Summary cards: totalFundIn, netPosition, office reported/real, personal, cash out, saving, remaining
- Transaction table: date, type, description, account, category, reported, real, saving
- Detail rows: note + receipt status ("Ada bukti" / "Belum ada bukti")
- Footer: "Generated by Funds Manager", page numbers

Output path: `{cacheDir}/reports/FundsManager_{project}_{timestamp}.pdf`

---

## Excel Contents

**Sheet 1 — Ringkasan:** project metadata + summary rows (same fields as PDF summary)

**Sheet 2 — Transaksi:** columns matching CSV + receipt status

Money cells stored as integer values with Rupiah number format.

---

## CSV Contents

Header row + transaction rows + `RINGKASAN` footer section.

Footer fields:
- Total Dana Masuk
- Dilaporkan ke Kantor
- Pengeluaran Real
- Expense Pribadi
- Selisih / Hemat
- Sisa Real
- Posisi Bersih

---

## Consistency Requirements

`CsvExportConsistencyTest` verifies CSV footer matches `CalculateProjectSummaryUseCase`.

Any export change must pass:
```bash
./gradlew testDebugUnitTest --tests "*.CsvExportConsistencyTest"
./gradlew testDebugUnitTest --tests "*.ExportCsvUseCaseTest"
```

## File Provider

Configured in `AndroidManifest.xml`:
- Authority: `${applicationId}.fileprovider`
- Paths: `@xml/file_paths`

Required for sharing PDF/XLSX from cache directory.
